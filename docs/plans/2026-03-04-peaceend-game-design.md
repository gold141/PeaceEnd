# PeaceEnd — Game Design Document

## Overview

**Жанр:** Реал-тайм артиллерийская стратегия, вид сбоку (2D)
**Движок:** Godot 4.6.1 (GDScript), экспорт в HTML5
**Стиль:** Пиксель-арт
**Платформа:** Web (Newgrounds и аналоги)
**Тема:** Современные войны

## Концепция

Игрок — командир огневой поддержки. Вид сбоку. Слева — база игрока, справа — база врага. Союзные юниты (пехота, техника) автоматически двигаются слева направо. Игрок управляет ТОЛЬКО артиллерией: выбирает оружие, задаёт угол и силу, стреляет. Снаряд летит по параболе в реальном времени — за время полёта враги могут сместиться. Задача — расчистить путь своим войскам и уничтожить вражескую базу.

## Ядро геймплея

### Роль игрока
1. Выбрать оружие из панели внизу экрана
2. Навести прицел — задать угол и силу (видна линия траектории)
3. Выстрелить — снаряд летит по параболе в реальном времени
4. Снаряд долетает через N секунд — за это время враги могут сместиться

### Автоматические союзные юниты
- Двигаются слева направо
- Встречая врага — стреляют сами
- Если впереди слишком сильный враг/укрепление — останавливаются и гибнут
- Задача игрока — уничтожить преграды ДО того, как юниты до них дойдут

### Условия
- **Победа:** уничтожить базу/штаб врага справа
- **Поражение:** враг уничтожает базу игрока слева, или все юниты игрока потеряны

## Система вооружения

| Оружие | Поведение | Перезарядка | Урон |
|--------|-----------|-------------|------|
| Миномёт | Крутая парабола, малая дальность | Быстрая | Средний, малый радиус |
| Гаубица | Классическая парабола, большая дальность | Средняя | Большой, большой радиус |
| РСЗО (Град) | Залп ракет, разброс | Долгая | Средний, широкая зона |
| Авиаудар | Самолёт горизонтально, бомба вертикально | Очень долгая | Огромный |
| Дрон-камикадзе | Рисуешь траекторию мышкой | Средняя | Точечный, высокий |
| Крылатая ракета | Прямая линия, задаёшь высоту | Очень долгая | Огромный, точечный |

### Механика прицеливания
- Миномёт/Гаубица/РСЗО: угол + сила (классические танчики)
- Авиаудар: выбрать точку сброса на горизонтали
- Дрон: рисуешь путь мышкой
- Ракета: указываешь цель, летит по прямой

## Юниты

### Союзные (автоматические)
| Юнит | Скорость | HP | Атака | Особенность |
|------|----------|----|-------|-------------|
| Пехота | Средняя | Низкий | Стрелковое | Дешёвые, много |
| БТР | Быстрая | Средний | Пулемёт | Перевозит пехоту |
| Танк | Медленная | Высокий | Пушка | Пробивает укрепления |
| Зенитка | Средняя | Средний | Зенитная ракета | Сбивает вражьи дроны/самолёты |

### Вражеские
| Объект | Тип | Опасность |
|--------|-----|-----------|
| Окоп с пехотой | Статичный | Убивает пехоту |
| Бункер | Статичный, бронированный | Блокирует продвижение |
| Танк | Подвижный | Уничтожает технику |
| ПВО | Статичный | Сбивает авиаудары |
| Вражеская артиллерия | Статичный | Стреляет по базе игрока |

**Ключевая механика:** Вражеская ПВО создаёт зоны запрета для авиации — нужно сначала уничтожить ПВО артиллерией.

## Структура игры
- Уровни/миссии с нарастающей сложностью
- Каждый уровень — новый набор врагов и условий
- Новое вооружение открывается по мере прогресса

## Физика
- Аркадная: простая парабола, без ветра
- Position = start + velocity * t + 0.5 * gravity * t²
- Без встроенной физики Godot — чистая математика в _process()
- Столкновения через Area2D

## MVP (первая версия)
- 1 тип артиллерии (миномёт)
- 1 тип союзного юнита (пехота)
- 2 типа врагов (окоп, бункер)
- 1 уровень
- Простой UI: кнопка стрельбы, индикатор угла/силы
- Нет меню (сразу в бой)
- Аркадная физика (простая парабола)
- Пиксель-арт графика
